# Security Policy

## Supported Versions

This is a personal blog and portfolio site. Security updates are applied to the latest version only.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly:

1. **Do not** open a public GitHub issue for security vulnerabilities
2. Email the details to **hi@gvns.ca**
3. Include:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Any suggested fixes (optional)

## Response Timeline

- **Acknowledgment**: Within 48 hours
- **Initial assessment**: Within 7 days
- **Resolution**: Depends on severity and complexity

## Scope

This policy covers:
- The gvns.ca website and its source code
- Build and deployment infrastructure
- Any APIs or services directly operated by gvns.ca

Out of scope:
- Third-party services (GitHub, Cloudflare, etc.)
- Social engineering attacks
- Denial of service attacks

## Recognition

Contributors who report valid security vulnerabilities will be acknowledged (with permission) in the project's security advisories.
